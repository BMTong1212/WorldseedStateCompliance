# WorldSeed State Compliance

Homeschool state compliance snapshot tool — part of the WorldSeed ecosystem.

🌱 **Rooted in faith. Grown for the world.**

## Features

- 5 researched state compliance profiles (VA, TX, FL, CA, MS)
- 3 source links per state for legal verification
- Print/Save PDF workflow for parent handouts
- Light & dark mode with WorldSeed brand theming
- Fully responsive for desktop, tablet, and mobile

## Tech Stack

- Single-file HTML application
- No build step required
- Vanilla CSS with custom properties
- Google Fonts (Cormorant Garamond, Nunito Sans, Patrick Hand)

## Deployment

Deployed via [Vercel](https://vercel.com) with automatic deploys from this GitHub repo.
